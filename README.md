# CAD Portfolio – Electrical Engineering Technology

## Overview
This portfolio showcases three CAD projects created using AI‑assisted modeling tools. Each design represents a practical component commonly used in electrical engineering applications, including enclosures, mounting hardware, and cable‑management solutions. All models were generated cleanly without requiring post‑processing in traditional CAD software.

The project are all presented in the form of STL files


## Project List

### 1. Electronics Enclosure with Standoffs and Cable Cutout
A compact rectangular enclosure designed for small embedded systems or sensor modules. Features include:
- Removable lid with aligned screw holes  
- Internal PCB standoffs  
- Side cable‑entry cutout  
This model matches the prompt and is fully portfolio‑ready.

### 2. PCB Standoff Spacer
A simple cylindrical spacer used for mounting printed circuit boards. Designed at a fixed height with a center mounting hole. This part is commonly used in prototyping, electronics assembly, and hardware mounting.

### 3. Wire Management Clip
A U‑shaped cable‑routing clip designed to hold multiple wires and mount to a flat surface using screw holes. Useful for organizing wiring in control panels, embedded systems, and lab setups.
